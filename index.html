The First Few Classes

Some Initial Thoughts

I'm excited to learn HTML and CSS mostly because I'm really interested in JavaScript and know I need to learn the basics before I learn how to make interactive web pages.

My background is in architecture and public art and I have helped produce interactive sculptures all over the country that use hardware like Arduino and Raspberry Pi to engage audiences. However, my background is not in coding so I have a lot to learn.

The first few classes have been not too challenging for me because I've already done a lot of experimenting on Treehouse and Codecademy prior to starting the class, but I know nothing about Git or Unix so those parts of the class have been really helpful.
